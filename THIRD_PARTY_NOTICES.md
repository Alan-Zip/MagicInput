# Third-party notices

## MagicTrackpad2ForWindows driver package

Magic Input's installer can download the Microsoft-signed Magic Trackpad 2
driver package from `vitoplantamura/MagicTrackpad2ForWindows`. This repository
does not vendor that driver package in git. Review the upstream project and
release package before installing it on your system.

## ThreeFingerDragOnWindows / RawInput.Touchpad approach

Parts of Magic Input's Precision Touchpad raw-input parsing are adapted from the
MIT-licensed ThreeFingerDragOnWindows project by Clement Grennerat, which itself
includes an edited RawInput.Touchpad implementation by emoacht.
